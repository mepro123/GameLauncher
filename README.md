## Installation Guide

1 - Run installation.bat as administrator
  - This executes the command "pip install PyQt6 psutil" which is the module that need this program to work
  - Creates a file in ProgramFiles named "Games" (Why we need administraitor privilages)
2 - Put your games into the folder (C:\Program Files\Games)
  - Game must have an EXE in it
3 - Run the program
